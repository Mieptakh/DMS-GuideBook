2. DASHBOARD

Dashboard adalah halaman utama setelah login. Fungsi:
  • Melihat jumlah data
  • Akses menu cepat
  • Melihat ringkasan sistem
---

3. AKSI CEPAT

Klik tombol untuk langsung menambah data:
  • Tambah Pendaftar
  • Tambah Program
  • Tulis Berita
  • Upload Foto
  • Tambah Prestasi
  • Tambah Testimoni
  • Tambah Reels
  • Tambah Admin
---

4. CARA MENAMBAH DATA (UMUM)

1. Klik menu atau tombol "Tambah"
2. Isi semua form yang tersedia
3. Klik tombol Simpan
4. Data akan masuk ke sistem
Pastikan data sudah benar sebelum disimpan!
---

5. CARA EDIT DATA

1. Masuk ke menu data (misal: Program / Berita)
2. Cari data yang ingin diubah
3. Klik tombol Edit
4. Ubah data sesuai kebutuhan
5. Klik Update / Simpan
---

6. CARA HAPUS DATA

1. Masuk ke menu data
2. Cari data yang ingin dihapus
3. Klik tombol Hapus
4. Klik OK / Konfirmasi
Data yang dihapus tidak bisa dikembalikan!
---
